# ME26 — SR24 Ren'Py 8.5.x Upgrade

This is a personal passion project with the goal of bringing **SR24** up to **Ren'Py 8.5.x** compatibility while keeping everything fully playable.

## Goals

- **v1.0** — A complete, fully working port of SR24 running on Ren'Py 8.5.x with no major regressions
- **Beyond 1.0** — Once the port is stable, the project may head in a creatively different direction

## Playtesters Welcome

Join the [Discord](https://discord.gg/7dr3xHRg4u) if you want to get involved or follow along.

If you're interested in helping test, bug reports and feedback are very much appreciated. Things to look out for:

- Crashes or tracebacks
- Broken dialogue, missing images, or audio issues
- Anything that behaved differently in the original SR24

Feel free to open an issue or get in touch directly.
